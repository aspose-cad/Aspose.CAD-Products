<p>
 Aspose.CAD für .NET ist eine eigenständige AutoCAD-Verarbeitungs-API. Es bietet die Funktionalität zum Konvertieren von DWG-, DWF- und DXF-Dateien in hochwertige PDF- und Rasterbilder. Entwickler können bestimmte Layouts und Ebenen aus den AutoCAD-Dateien auswählen und konvertieren sowie den gesamten Dateikonvertierungsprozess mit Leichtigkeit verfolgen.
</p>

    <h2 class="h2title">
     Konvertieren Sie CAD in PDF
    </h2>
    <p>
     Aspose.CAD für .NET bietet die Möglichkeit, unterstützte CAD-Zeichnungen in das PDF-Format zu konvertieren. Der Konvertierungsprozess ist hochgradig anpassbar, sodass Sie die Rendering-Aspekte wie Seitengröße, Hintergrundfarbe, Zeichenfarbe, Skalierungsmodus usw. konfigurieren können.
    </p>
    <div class="codeblock" id="code">
     <h3>
      CAD im PDF-Format speichern - C#
     </h3>
     <pre><code class="cs">// Laden Sie die zu konvertierende Zeichnung

using (var img = Aspose.CAD.Image.Load(dir + "template.dwg"))

{

    // Erstellen Sie eine Instanz von CadRasterizationOptions &amp; Legen Sie die resultierende Seitengröße fest

    var rasterizationOptions = new Aspose.CAD.ImageOptions.CadRasterizationOptions()

    {

        PageSize = new Aspose.CAD.SizeF(1200, 1200),

    };

    // Speichern Sie das resultierende PDF

    img.Save(dir + "output.pdf", new Aspose.CAD.ImageOptions.PdfOptions() { VectorRasterizationOptions = rasterizationOptions });

}</code></pre>
