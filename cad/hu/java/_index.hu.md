<p>
 Az Aspose.CAD for Java egy fejlett AutoCAD konverziós natív API, szoftverfüggőségek nélkül. Lehetővé teszi a fejlesztők számára a DWF, DWG és DXF fájlok konvertálását raszteres képformátumokká és PDF formátumokká. Az átalakítási folyamat során a fejlesztők választhatnak és konvertálhatnak meghatározott rétegeket és elrendezéseket, valamint nyomon követhetik a fájlkonverziós folyamatot.
</p>

    <div class="codeblock" id="code">
     <h3>
      CAD mentése PDF formátumban - Java
     </h3>
     <pre><code class="java">// töltse be a CAD-t az átalakításhoz

Image image = Image.load(dir + "template.dwg");

// példány létrehozása &amp; állítsa be az eredményül kapott oldalméretet

CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();

rasterizationOptions.setPageSize(new SizeF(1200, 1200));

// PDF mentése

PdfOptions pdfOptions = new PdfOptions();

pdfOptions.setVectorRasterizationOptions(rasterizationOptions);

image.save(dir + "output.pdf", pdfOptions);

</code></pre>
