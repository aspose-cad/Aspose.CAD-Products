<p>
 Aspose.CAD para Java es una API nativa de conversión avanzada de AutoCAD sin dependencias de software. Permite a los desarrolladores la conversión de archivos DWF, DWG y DXF a formatos de imagen raster y PDF. Durante el proceso de conversión, los desarrolladores pueden elegir y convertir capas y diseños específicos, así como realizar un seguimiento del proceso de conversión de archivos.
</p>

    <h2 class="h2title">
     Conversión de formato CAD a PDF
    </h2>
    <p>
     Aspose.CAD para la API de Java admite la conversión de dibujos CAD a PDF. Los desarrolladores pueden ajustar fácilmente las opciones de dibujo, como el color de dibujo, el color de fondo, el tamaño de página, el modo de escala, etc., antes de la conversión.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Guardar CAD en formato PDF - Java
     </h3>
     <pre><code class="java">// cargar CAD para conversión

Image image = Image.load(dir + "template.dwg");

// crear instancia &amp; establecer el tamaño de página resultante

CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();

rasterizationOptions.setPageSize(new SizeF(1200, 1200));

// guardar PDF

PdfOptions pdfOptions = new PdfOptions();

pdfOptions.setVectorRasterizationOptions(rasterizationOptions);

image.save(dir + "output.pdf", pdfOptions);

</code></pre>
    </div>
