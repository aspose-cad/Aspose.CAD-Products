    <h2 class="h2title">
     Convertir CAD a PDF
    </h2>
    <p>
     Aspose.CAD para .NET brinda la capacidad de convertir dibujos CAD admitidos a formato PDF. El proceso de conversión es altamente personalizable, lo que le permite configurar los aspectos de representación, como el tamaño de página, el color de fondo, el color de dibujo, el modo de escala, etc.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Guardar CAD en formato PDF - C#
     </h3>
     <pre><code class="cs">// cargar el dibujo a convertir

using (var img = Aspose.CAD.Image.Load(dir + "template.dwg"))

{

    // crear una instancia de CadRasterizationOptions &amp; establecer el tamaño de página resultante

    var rasterizationOptions = new Aspose.CAD.ImageOptions.CadRasterizationOptions()

    {

        PageSize = new Aspose.CAD.SizeF(1200, 1200),

    };

    // guardar el PDF resultante

    img.Save(dir + "output.pdf", new Aspose.CAD.ImageOptions.PdfOptions() { VectorRasterizationOptions = rasterizationOptions });

}</code></pre>
    </div>
