 Aspose.CAD per Java è un'API nativa di conversione AutoCAD avanzata senza dipendenze software. Consente agli sviluppatori la conversione di file DWF, DWG e DXF in formati di immagini raster e PDF.     <h2 class="h2title">
     Conversione da CAD a PDF
    </h2>
    <p>
     Aspose.CAD per Java API supporta la conversione di disegni CAD in PDF. Gli sviluppatori possono facilmente regolare le opzioni di disegno come il colore del disegno, il colore dello sfondo, le dimensioni della pagina, la modalità di ridimensionamento e così via prima della conversione.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Salva CAD in formato PDF - Java
     </h3>
     <pre><code class="java">// caricare CAD per la conversione

Image image = Image.load(dir + "template.dwg");

// crea istanza &amp; impostare la dimensione della pagina risultante

CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();

rasterizationOptions.setPageSize(new SizeF(1200, 1200));

// salva PDF

PdfOptions pdfOptions = new PdfOptions();

pdfOptions.setVectorRasterizationOptions(rasterizationOptions);

image.save(dir + "output.pdf", pdfOptions);

</code></pre>
    </div>
