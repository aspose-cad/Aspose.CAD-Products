 Aspose.CAD для .NET — это автономный API обработки AutoCAD. Он предоставляет функциональные возможности для преобразования файлов DWG, DWF и DXF в высококачественные PDF и растровые изображения.     </h2>
    <p>
     Aspose.CAD для .NET предоставляет возможность конвертировать поддерживаемые чертежи САПР в формат PDF. Процесс преобразования легко настраивается, что позволяет настраивать аспекты рендеринга, такие как размер страницы, цвет фона, цвет отрисовки, режим масштабирования и т. д.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Сохранение САПР в формате PDF — C#
     </h3>
     <pre><code class="cs">// загрузить чертеж для преобразования

using (var img = Aspose.CAD.Image.Load(dir + "template.dwg"))

{

    // создать экземпляр CadRasterizationOptions &amp; установить результирующий размер страницы

    var rasterizationOptions = new Aspose.CAD.ImageOptions.CadRasterizationOptions()

    {

        PageSize = new Aspose.CAD.SizeF(1200, 1200),

    };

    // сохранить полученный PDF

    img.Save(dir + "output.pdf", new Aspose.CAD.ImageOptions.PdfOptions() { VectorRasterizationOptions = rasterizationOptions });

}</code></pre>
