<p>
 Aspose.CAD pour Java est une API native de conversion AutoCAD avancée sans aucune dépendance logicielle. Il permet aux développeurs de convertir des fichiers DWF, DWG et DXF en formats d'image raster et PDF. Pendant le processus de conversion, les développeurs peuvent choisir et convertir des calques et des mises en page spécifiques, ainsi que suivre le processus de conversion des fichiers.
</p>

    <h2 class="h2title">
     Conversion de format CAO en PDF
    </h2>
    <p>
     L'API Aspose.CAD pour Java prend en charge la conversion de dessins CAO en PDF. Les développeurs peuvent facilement ajuster les options de dessin telles que la couleur de dessin, la couleur d'arrière-plan, la taille de la page, le mode d'échelle, etc. avant la conversion.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Enregistrer la CAO au format PDF - Java
     </h3>
     <pre><code class="java">// charger CAD pour la conversion

Image image = Image.load(dir + "template.dwg");

// créer une instance &amp; définir la taille de la page résultante

CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();

rasterizationOptions.setPageSize(new SizeF(1200, 1200));

// enregistrer le PDF

PdfOptions pdfOptions = new PdfOptions();

pdfOptions.setVectorRasterizationOptions(rasterizationOptions);

image.save(dir + "output.pdf", pdfOptions);

</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
