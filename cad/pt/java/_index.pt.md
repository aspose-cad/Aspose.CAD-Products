<p>
 Aspose.CAD para Java é uma API nativa de conversão avançada do AutoCAD sem dependências de software. Ele permite aos desenvolvedores a conversão de arquivos DWF, DWG e DXF para formatos de imagem raster e PDF. Durante o processo de conversão, os desenvolvedores podem escolher e converter camadas e layouts específicos, bem como acompanhar o processo de conversão de arquivos.
</p>

    </h2>
    <p>
     Aspose.CAD para API Java suporta a conversão de desenhos CAD para PDF. Os desenvolvedores podem ajustar facilmente as opções de desenho, como cor de desenho, cor de fundo, tamanho da página, modo de escala e assim por diante, antes da conversão.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Salvar CAD em formato PDF - Java
     </h3>
     <pre><code class="java">// carregar CAD para conversão

Image image = Image.load(dir + "template.dwg");

// criar instância &amp; definir o tamanho da página resultante

CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();

rasterizationOptions.setPageSize(new SizeF(1200, 1200));

// salvar PDF

PdfOptions pdfOptions = new PdfOptions();

pdfOptions.setVectorRasterizationOptions(rasterizationOptions);

image.save(dir + "output.pdf", pdfOptions);

</code></pre>
