    <h2 class="h2title">
     Converter CAD para PDF
    </h2>
    <p>
     Aspose.CAD para .NET fornece a capacidade de converter desenhos CAD suportados para o formato PDF. O processo de conversão é altamente personalizável, permitindo configurar os aspectos de renderização, como tamanho da página, cor do plano de fundo, cor do desenho, modo de escala e assim por diante.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Salvar CAD em formato PDF - C#
     </h3>
     <pre><code class="cs">// carregue o desenho a ser convertido

using (var img = Aspose.CAD.Image.Load(dir + "template.dwg"))

{

    // crie uma instância de CadRasterizationOptions &amp; definir o tamanho da página resultante

    var rasterizationOptions = new Aspose.CAD.ImageOptions.CadRasterizationOptions()

    {

        PageSize = new Aspose.CAD.SizeF(1200, 1200),

    };

    // salvar PDF resultante

    img.Save(dir + "output.pdf", new Aspose.CAD.ImageOptions.PdfOptions() { VectorRasterizationOptions = rasterizationOptions });

}</code></pre>
    </div>
