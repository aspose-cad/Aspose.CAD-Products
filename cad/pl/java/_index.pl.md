    <p>
     Aspose.CAD dla Java API obsługuje konwersję rysunków CAD do formatu PDF. Deweloperzy mogą łatwo dostosować opcje rysowania, takie jak kolor rysowania, kolor tła, rozmiar strony, tryb skalowania i tak dalej przed konwersją.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Zapisz CAD w formacie PDF - Java
     </h3>
     <pre><code class="java">// wczytaj CAD do konwersji

Image image = Image.load(dir + "template.dwg");

// utwórz instancję &amp; ustaw wynikowy rozmiar strony

CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();

rasterizationOptions.setPageSize(new SizeF(1200, 1200));

// zapisz PDF

PdfOptions pdfOptions = new PdfOptions();

pdfOptions.setVectorRasterizationOptions(rasterizationOptions);

image.save(dir + "output.pdf", pdfOptions);

</code></pre>
